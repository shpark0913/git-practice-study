## 0802_codeproblem

- 개요 : 1부터 주어진 횟수까지 2를 곱한 값(들)을 출력하시오.

- 제약 : 각 수는 30 이하

- 입력 : 30 이하의 숫자 입력 (ex : 8)

- 출력 : 1 2 4 8 16 32 64 128 256





## 0802_codeproblem2

- 개요 : 369게임 만들기

- 제약 : 
  
  - 1부터 입력한 n까지 숫자를 차례대로 말한다.
  
  - 3, 6, 9 가 들어가 있는 수는 말하지 않고 - 로 대신한다.
  
  - 33과 같은 수는 --로 대신한다.
  
  - n 은 100 이하의 자연수로 한다.

- 입력 : 정수 n  (ex : 9)

- 출력 : 1 2 - 4 5 - 7 8 -
